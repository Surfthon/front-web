## 💬 소개

교통사고는 전 세계적으로 큰 문제이며, 이를 예방하기 위한 다양한 노력이 기울여지고 있습니다.    
이 프로젝트의 목표는 빅 데이터 분석을 활용하여 교통사고 발생 가능성을 예측하고, 이를 통해 사고를 예방하는 시스템을 구축하는 것입니다.
